Aplicando lo aprendido

Laboratorio II

Abril 25, 2022 Dayana Aymara

Este inicio de clases fue virtual. Aprendí muchos nuevos conceptos relacionados a HTML y CSS. Además, me familiaricé con una nueva aplicación que descargamos "Visual Studio Code" que nos servirá para la creación de nuestra web con ayuda de algunos templates. Relacionarse con algunos conceptos fue algo complicado al inicio, pero con práctica todo resultó muy bien.

Visual Code, la aplicación que empleamos, me ha ayudado mucho a entender el funcionamiento interno de una página web. Es bastante interesante y divertido cuando le agarras el ritmo a todo. Lo recomendable es anotar todo lo posible para que después se pueda aplicar sin ningún problema en la aplicación. Eso nos va a facilitar mucho el aprendizaje.
